# Styles

스타일 관련 파일들을 관리하는 폴더입니다.

## 구조

- `globals.css` - 전역 스타일
- `variables.css` - CSS 변수들
- `components/` - 컴포넌트별 스타일
- `themes/` - 테마 관련 스타일

## 지원하는 스타일링 방법

- CSS Modules
- Styled Components
- Tailwind CSS
- CSS-in-JS
- Sass/SCSS

## 네이밍 컨벤션

- 파일명: kebab-case (예: `button.module.css`, `header-styles.css`)
- CSS 클래스명: kebab-case (예: `.button-primary`, `.header-container`)
